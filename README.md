1. Instalar librerías: pip install -r requirements.txt

2. Ejecución rápida: Doble clic en ejecutar_bot.bat

3. Ejecución manual: python main.py 
        (Asegúrese de tener el entorno virtual activo. Si no sabe cómo, ejecute el comando de activación correspondiente a su sistema):

        Windows: .\venv\Scripts\activate

        Mac/Linux: source venv/bin/activate
